# 🧠 HR Attrition AI Analyst

> Ask questions about employee attrition in plain English — get AI-powered answers instantly.

🔴 **[Live Demo →](https://hr-attrition-ai-analyst.streamlit.app/)**

---

## What it does
- 💬 Chat with your HR data using natural language
- 📊 5 interactive charts — department, income, overtime, age, job role
- 🎯 5 live KPI cards — attrition rate, income gap, overtime risk
- 📋 One-click executive summary generation
- 🧠 Powered by Llama3-70b via Groq API

## Tech Stack
`Python` `Streamlit` `Groq API` `Llama3` `Pandas` `Plotly`
